List of Passwords[]

Similar to the Tamagotchi Connection Version 3 and Keitai Kaitsuu Tamagotchi Plus, the Tamagotchi Connection features a password system that can unlock food and items on the device. These passwords are found on the official Tamagotchi Connection website, included in the packaging of shells, and given away during promotional collaborations.

List of Secret Codes[]

The Tamagotchi Connection also features secret codes which also unlock food and items on the device. Each is free upon first entering a code, though all except for the Stuffed Tama 2 (Hohotchi Costume) can be purchased when entering the code again.

Name Sprite Code Gotchi Points Additional Details
Cake
ABBAACBA 600 Source: Included in manual

Can also be purchased on the entered birthday and December 25th (Christmas Day).
Steak
BCABACBC 800 Source:

Liked by all characters.

Cuckoo Clock
CBACCABC 4000 Source: Obtained by clicking on Nikatchi on the main page of Tamagotchi Connection website

Has a special animation upon the turn of the hour.
Hair Gel
ACBBBACC 400 Source: Obtained by clicking on Oyajitchi on the main page of Tamagotchi Connection website

Only adult characters can use it.
Love Potion
BCBCCABA 2000 Source: Obtained by clicking on Pipotchi on the main page of Tamagotchi Connection website

Also known as Honey. Only adult characters can use it, and will automatically max out the relationship meter of the next Tamagotchi it connects with.
RC Car
AABBCACB 2800 Source:
Stuffed Tama 1 (Nyatchi Costume)
CACABABC 2500 Source:

Worn by the character but can only be used by adults. It is taken off when the character goes to bed.

Stuffed Tama 2 (Hohotchi Costume)
N/A N/A Worn by the character but can only be used by adults. It is taken off when the character goes to bed. This item does not have its own code but instead replaces the item for the final code entered. It can only be obtained once, but can be used multiple times.
